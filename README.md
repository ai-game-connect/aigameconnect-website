# AI Game Connect Website

This is the main website for AI Game Connect.

AI Game Connect is the parent brand: an AI automation and gaming innovation company for real-life gaming communities, venue networks, rankings, competitions, and smart operations.

Dawrak / دورك is the first product under AI Game Connect. It focuses on real-life player matchmaking in approved venues.

Senet is the competition and tournament system under AI Game Connect. It starts with local and area competitions, then grows toward larger tournament experiences.

## Build

This is a simple static website designed to stay lean and GitHub Pages friendly.

Main files:

- `index.html`
- `styles.css`
- `script.js`
- `assets/`

There is no framework, build system, backend, database, or account system.
